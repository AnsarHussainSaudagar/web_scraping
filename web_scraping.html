Web Scraping with Python

Web scraping is a technique used to extract data from websites using Python libraries like BeautifulSoup and requests.

Example Code:

import requests
from bs4 import BeautifulSoup

url = "https://example.com"
response = requests.get(url)
soup = BeautifulSoup(response.text, "html.parser")
print(soup.title.text)

This script fetches a webpage and extracts the title.
